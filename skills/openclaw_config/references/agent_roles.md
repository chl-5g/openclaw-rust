# Agent 角色说明

## 角色类型

### Executive (高管)
- **CEO**: 负责整体规划和最终决策
- **COO**: 负责运营协调

### Management (管理)
- **Product Manager**: 产品规划和管理
- **Project Manager**: 项目管理

### Execution (执行)
- **Engineer**: 开发工程师
- **Designer**: 设计师

### Validation (验证)
- **QA**: 质量保证
- **Reviewer**: 代码审查

## 能力定义

每个角色可以有不同的能力：
- `planning`: 规划能力
- `delegation`: 任务分发能力
- `analysis`: 分析能力
- `coding`: 编码能力
- `testing`: 测试能力
- `review`: 审核能力
